How to Use the Recognizer

Follow these steps to use the voice command recognizer:

  • Click on 'Start Recognition' to enable voice recognition and allow microphone access.
  • Say a number between zero (home) and ten.
  • If the number is recognized, a popup will ask if you want to navigate to the corresponding page.
  • Say 'yes' to navigate, or 'no' to stay on the current page.
